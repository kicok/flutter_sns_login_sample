## Getting Started

fire base 프로젝트 생성

- [fire base](https://console.firebase.google.com/)

  - ios 앱추가

    1. 오류시 ios/Podfile 수정 [#platform :ios, '9.0'] 을 [platform :ios, '10.0']으로 코드 변경 이후
    2. 터미널에서 cd ios 로 디렉토리 변경하고
    3. pod update 하면 된다.

  - android 앱추가
